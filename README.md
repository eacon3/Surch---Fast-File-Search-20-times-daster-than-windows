# Surch FastFileSearch-20-times-daster-than-windows
Surch is a lightning-fast file search tool I built to help find files on your computer quickly and easily. It's designed to be way faster and more efficient than the default Windows search, using modern programming techniques to make the most of your system's resources
**Indexing**
More than 20 times faster than Windows
## How to Run Surch
1. **Build the project**: `cargo build --release`
2. **Run the application**: `cargo run --release`
